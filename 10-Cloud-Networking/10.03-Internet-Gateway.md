# Internet Gateway (IGW)

## What is Internet Gateway?

Internet Gateway is a component that allows communication between a VPC and the Internet.

Without an Internet Gateway, resources cannot access the Internet.

## Functions

- Incoming Internet Traffic
- Outgoing Internet Traffic
- Route Internet Packets

## Architecture

Internet
    │
    ▼
Internet Gateway
    │
    ▼
Public Subnet
    │
    ▼
EC2 Instance

## How It Works

Step 1:
User requests a website.

Step 2:
Traffic reaches Internet Gateway.

Step 3:
Gateway forwards traffic to instance.

Step 4:
Response returns through gateway.

## Route Table Entry

Destination:
0.0.0.0/0

Target:
Internet Gateway

Example:

0.0.0.0/0 → igw-123456

## Key Points

- Attached to VPC
- One IGW per VPC
- Highly Available
- Managed Service

## Interview Question

Q: Can a public subnet work without Internet Gateway?

A:

No. A public subnet requires an Internet Gateway and proper route table configuration.
