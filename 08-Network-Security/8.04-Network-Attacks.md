# Common Network Attacks

## What is a Network Attack?

A network attack is an attempt to gain unauthorized access, disrupt services, steal data, or compromise systems.

---

# 1. DoS Attack

Denial of Service

Goal:

Make service unavailable.

Example:

Sending millions of requests to a server.

---

# 2. DDoS Attack

Distributed Denial of Service

Attack comes from multiple systems simultaneously.

More dangerous than DoS.

---

# 3. Man-In-The-Middle (MITM)

Attacker secretly intercepts communication.

User ↔ Attacker ↔ Server

Can steal:

- Passwords
- Sensitive data

---

# 4. Phishing Attack

Fake websites or emails.

Goal:

Steal credentials.

Example:

Fake banking website.

---

# 5. Spoofing

Attacker impersonates trusted device.

Types:

- IP Spoofing
- ARP Spoofing
- DNS Spoofing

---

# 6. DNS Attack

Manipulates DNS records.

Redirects users to malicious websites.

---

# 7. Brute Force Attack

Tries multiple password combinations.

Goal:

Gain unauthorized access.

---

# 8. Malware Attack

Malicious software.

Types:

- Virus
- Worm
- Trojan
- Ransomware

---

# 9. Packet Sniffing

Capturing network packets.

Tools:

- Wireshark
- Tcpdump

---

# 10. Session Hijacking

Attacker steals session information.

Can gain access to authenticated sessions.

---

# Prevention Methods

- Firewall
- VPN
- HTTPS
- MFA
- Strong Passwords
- IDS/IPS
- Security Monitoring
