# Ports and Protocols

A Port is a logical communication endpoint.

IP Address identifies:

Which Device?

Port Number identifies:

Which Application?

---

# Example

IP Address

192.168.1.10

Port

80

Meaning:

Send data to web server running on that machine.

---

# Common Port Numbers

| Port | Protocol | Purpose |
|---------|---------|---------|
| 20 | FTP Data | File Transfer |
| 21 | FTP Control | File Transfer |
| 22 | SSH | Secure Login |
| 23 | Telnet | Remote Login |
| 25 | SMTP | Email Sending |
| 53 | DNS | Name Resolution |
| 67 | DHCP | Server |
| 68 | DHCP | Client |
| 80 | HTTP | Website |
| 110 | POP3 | Email |
| 143 | IMAP | Email |
| 443 | HTTPS | Secure Website |
| 3306 | MySQL | Database |
| 5432 | PostgreSQL | Database |
| 6379 | Redis | Cache |
| 27017 | MongoDB | Database |
| 8080 | Alternate HTTP | Web Apps |

# Important Protocols

## HTTP

Hyper Text Transfer Protocol

Port: 80

Used for websites.

---

## HTTPS

Secure HTTP

Port: 443

Uses SSL/TLS encryption.

---

## FTP

File Transfer Protocol

Ports:

20, 21

Used for file transfer.

---

## SSH

Secure Shell

Port: 22

Used for remote server access.

---

## SMTP

Simple Mail Transfer Protocol

Port: 25

Used to send emails.

---

## DNS

Domain Name System

Port: 53

Converts:

google.com

↓

142.250.xxx.xxx
