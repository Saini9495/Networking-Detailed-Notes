# Encapsulation and Decapsulation

These are the most important networking concepts.

Interviewers ask this very frequently.

---

# What is Encapsulation?

Process of adding headers while data travels from sender to receiver.

As data moves down the TCP/IP layers:

Extra information gets added.

This process is called:

Encapsulation

---

# Example

User sends:

Hello

Application Layer

Data = Hello

↓

Transport Layer

Adds TCP Header

↓

Internet Layer

Adds IP Header

↓

Network Access Layer

Adds MAC Header

↓

Sent over network

# Protocol Data Units (PDU)

Each layer uses different names.

| Layer | PDU |
|---------|---------|
| Application | Data |
| Transport | Segment |
| Internet | Packet |
| Network Access | Frame |

---

# What is Decapsulation?

Receiver removes headers one by one.

This process is called:

Decapsulation

---

# Real-Life Example

Sending a Courier

Letter = Data

Envelope = TCP Header

Address Label = IP Header

Delivery Sticker = MAC Header

Courier Company = Network

Receiver removes all layers and reads the letter.

This is Decapsulation.

---

# Interview Question

Q: Which layer adds IP Address?

Answer:

Internet Layer

---

Q: Which layer adds MAC Address?

Answer:

Network Access Layer

---

Q: Which layer adds Port Number?

Answer:

Transport Layer

---

# Summary

Application Layer → Data

Transport Layer → Segment

Internet Layer → Packet

Network Access Layer → Frame

Frame travels across network and reaches destination.

Destination performs Decapsulation and retrieves original data.
