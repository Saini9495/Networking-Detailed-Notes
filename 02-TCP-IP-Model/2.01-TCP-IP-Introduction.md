# TCP/IP Model

TCP/IP stands for:

Transmission Control Protocol / Internet Protocol

TCP/IP is the standard communication model used on the Internet.

Whenever you:

- Open a website
- Send an email
- Watch YouTube
- Use WhatsApp

TCP/IP is working behind the scenes.

---

# Why TCP/IP?

Different devices use different operating systems:

- Windows
- Linux
- MacOS
- Android

TCP/IP provides a common language for communication.

---

# TCP/IP Layers

TCP/IP Model consists of 4 layers:

| Layer | Function |
|---------|---------|
| Application | User Interaction |
| Transport | End-to-End Communication |
| Internet | Routing & IP Addressing |
| Network Access | Physical Transmission |

---

# TCP/IP vs OSI Model

| OSI | TCP/IP |
|---------|---------|
| Application | Application |
| Presentation | Application |
| Session | Application |
| Transport | Transport |
| Network | Internet |
| Data Link | Network Access |
| Physical | Network Access |

---

# Real-Life Example

Sending a Parcel

Application Layer → Write letter

Transport Layer → Divide into packages

Internet Layer → Add destination address

Network Access Layer → Deliver through roads

Receiver gets parcel and opens it.
