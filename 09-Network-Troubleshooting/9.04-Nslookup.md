# Nslookup Command

## What is Nslookup?

Nslookup is used to query DNS servers and obtain domain name or IP address information.

## Purpose

- DNS troubleshooting
- Domain resolution testing
- Verify DNS records

## Basic Usage

```bash
nslookup google.com
```

Output:

```bash
Name: google.com
Address: 142.250.193.78
```

## Reverse Lookup

```bash
nslookup 8.8.8.8
```

## Query Specific DNS Server

```bash
nslookup google.com 8.8.8.8
```

## Check MX Records

```bash
nslookup
set type=mx
google.com
```

## Common Record Types

| Record | Purpose |
|----------|---------|
| A | IPv4 Address |
| AAAA | IPv6 Address |
| MX | Mail Server |
| CNAME | Alias |
| TXT | Text Record |

## Real World Example

Verify if a website domain resolves correctly:

```bash
nslookup example.com
```
