# Tcpdump Command

## What is Tcpdump?

Tcpdump is a packet analyzer used to capture and inspect network traffic in real time.

It is one of the most important networking troubleshooting tools.

## Installation

Ubuntu:

```bash
sudo apt install tcpdump
```

## Basic Capture

```bash
sudo tcpdump
```

## Capture on Specific Interface

```bash
sudo tcpdump -i eth0
```

## Capture Specific Host

```bash
sudo tcpdump host 192.168.1.10
```

## Capture Specific Port

```bash
sudo tcpdump port 80
```

## Capture TCP Traffic

```bash
sudo tcpdump tcp
```

## Capture UDP Traffic

```bash
sudo tcpdump udp
```

## Save Packets to File

```bash
sudo tcpdump -w capture.pcap
```

## Read Capture File

```bash
sudo tcpdump -r capture.pcap
```

## Useful Filters

### Source IP

```bash
sudo tcpdump src 192.168.1.10
```

### Destination IP

```bash
sudo tcpdump dst 192.168.1.20
```

### Multiple Conditions

```bash
sudo tcpdump host 192.168.1.10 and port 443
```

## Real World Examples

### Monitor HTTP Traffic

```bash
sudo tcpdump port 80
```

### Monitor HTTPS Traffic

```bash
sudo tcpdump port 443
```

### Debug Kubernetes Network Issue

```bash
sudo tcpdump -i eth0 host <pod-ip>
```

## Benefits

- Packet-level visibility
- Troubleshoot latency
- Analyze protocols
- Debug application connectivity
