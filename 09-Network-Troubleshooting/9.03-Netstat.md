# Netstat Command

## What is Netstat?

Netstat displays active network connections, listening ports, routing tables, and interface statistics.

## Why Use Netstat?

- View open ports
- Check active connections
- Identify services running
- Troubleshoot networking issues

## Basic Commands

### Show All Connections

```bash
netstat -a
```

### Show Listening Ports

```bash
netstat -l
```

### Show Numeric Addresses

```bash
netstat -n
```

### Show Process Information

```bash
netstat -p
```

### Combined Command

```bash
netstat -tulnp
```

## Meaning of Flags

| Flag | Description |
|--------|-------------|
| t | TCP |
| u | UDP |
| l | Listening |
| n | Numeric |
| p | Process |

## Example Output

```bash
tcp 0 0 0.0.0.0:22 LISTEN
```

Meaning:

- TCP Protocol
- Port 22
- SSH Service Listening

## Real World Example

Check if Nginx is running:

```bash
netstat -tulnp | grep 80
```

## Modern Alternative

```bash
ss -tulnp
```

ss is faster than netstat.
