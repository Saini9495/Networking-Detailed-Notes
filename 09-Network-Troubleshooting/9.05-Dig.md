# Dig Command

## What is Dig?

Dig (Domain Information Groper) is an advanced DNS lookup utility.

It provides detailed DNS information.

## Basic Syntax

```bash
dig google.com
```

## Example Output

```bash
;; ANSWER SECTION:
google.com. 300 IN A 142.250.193.78
```

## Useful Commands

### Query A Record

```bash
dig google.com A
```

### Query MX Record

```bash
dig google.com MX
```

### Query TXT Record

```bash
dig google.com TXT
```

### Query Name Server

```bash
dig google.com NS
```

### Short Output

```bash
dig google.com +short
```

## Check Specific DNS Server

```bash
dig @8.8.8.8 google.com
```

## Benefits Over Nslookup

- More detailed output
- Better DNS troubleshooting
- Preferred by Linux administrators

## Real World Example

Troubleshoot DNS issue:

```bash
dig mydomain.com
```
