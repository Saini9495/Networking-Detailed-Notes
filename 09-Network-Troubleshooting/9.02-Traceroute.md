# Traceroute Command

## What is Traceroute?

Traceroute shows the path packets take from source to destination.

It helps identify where network delays or failures occur.

## Commands

Linux:

```bash
traceroute google.com
```

Windows:

```cmd
tracert google.com
```

## How It Works

Traceroute sends packets with increasing TTL values.

Example:

TTL=1
TTL=2
TTL=3

Each router decreases TTL by 1.

When TTL becomes 0, router returns an ICMP message.

This reveals every hop.

## Example Output

```bash
1 192.168.1.1
2 10.10.0.1
3 172.16.1.1
4 google.com
```

## Benefits

- Detect routing issues
- Find network bottlenecks
- Troubleshoot packet loss

## Real World Example

Website loading slowly:

```bash
traceroute google.com
```

Check which network hop causes delay.
