# Ping Command

## What is Ping?

Ping is a network troubleshooting utility used to check whether a host is reachable over a network.

It sends ICMP Echo Request packets to the destination and waits for ICMP Echo Reply packets.

## Why Use Ping?

- Check network connectivity
- Verify host availability
- Measure latency
- Troubleshoot network issues

## Syntax

```bash
ping <hostname>
```

Example:

```bash
ping google.com
```

## Sample Output

```bash
PING google.com (142.250.193.78): 56 data bytes
64 bytes from 142.250.193.78: icmp_seq=0 ttl=116 time=15 ms
```

## Important Fields

### icmp_seq

Packet sequence number.

### ttl

Time To Live value.

### time

Round-trip time taken by packet.

## Common Options

### Linux

```bash
ping -c 4 google.com
```

Send only 4 packets.

### Windows

```cmd
ping -n 4 google.com
```

## Real World Use Case

Check if a server is reachable:

```bash
ping 192.168.1.10
```

## Limitations

- Some servers block ICMP.
- Successful ping does not guarantee application availability.
