# Network and Broadcast Address

## Network Address

The first IP address in a subnet.

Used to identify the network itself.

Cannot be assigned to devices.

Example:

Network:
192.168.1.0/24

Network Address:
192.168.1.0

---

## Broadcast Address

The last IP address in a subnet.

Used to communicate with all devices in subnet.

Cannot be assigned to devices.

Example:

Network:
192.168.1.0/24

Broadcast Address:
192.168.1.255

---

## Host Range

Usable IPs between:

Network Address
and
Broadcast Address

Example:

Network:
192.168.1.0

Broadcast:
192.168.1.255

Usable:

192.168.1.1

to

192.168.1.254

---

## Quick Formula

Network Address:

All Host Bits = 0

Broadcast Address:

All Host Bits = 1

## Interview Question

Q: Can Network Address be assigned to a host?

Answer:
No. Network address identifies the subnet itself.
