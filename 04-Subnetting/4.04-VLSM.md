# VLSM (Variable Length Subnet Mask)

## What is VLSM?

VLSM allows subnetting a network into subnets of different sizes.

Unlike traditional subnetting, VLSM provides flexibility and reduces IP wastage.

---

## Why VLSM?

Suppose you have:

Department A = 100 Hosts

Department B = 50 Hosts

Department C = 20 Hosts

Using same subnet size wastes IP addresses.

VLSM solves this problem.

---

## Example Network

Network:

192.168.1.0/24

Requirements:

A = 100 Hosts

B = 50 Hosts

C = 20 Hosts

---

### Department A

Required:

100 Hosts

Nearest:

128 Addresses

CIDR:

/25

Subnet:

192.168.1.0/25

Host Range:

192.168.1.1 - 192.168.1.126

Broadcast:

192.168.1.127

---

### Department B

Required:

50 Hosts

Nearest:

64 Addresses

CIDR:

/26

Subnet:

192.168.1.128/26

Host Range:

192.168.1.129 - 192.168.1.190

Broadcast:

192.168.1.191

---

### Department C

Required:

20 Hosts

Nearest:

32 Addresses

CIDR:

/27

Subnet:

192.168.1.192/27

Host Range:

192.168.1.193 - 192.168.1.222

Broadcast:

192.168.1.223

---

## Advantages

- Better IP utilization
- Reduced wastage
- Flexible design
- Scalable network

---

## Interview Question

Q: Difference between FLSM and VLSM?

FLSM:
All subnets same size.

VLSM:
Subnets can have different sizes.
