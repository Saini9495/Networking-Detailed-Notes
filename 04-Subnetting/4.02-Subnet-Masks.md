# Subnet Masks

## What is a Subnet Mask?

A subnet mask is used to identify which part of an IP address belongs to:

- Network Portion
- Host Portion

Example:

IP Address:
192.168.1.10

Subnet Mask:
255.255.255.0

Network:
192.168.1

Host:
10

---

## Binary Representation

IP:

192.168.1.10

11000000.10101000.00000001.00001010

Mask:

255.255.255.0

11111111.11111111.11111111.00000000

1 = Network Bit

0 = Host Bit

---

## CIDR Notation

CIDR means:

Classless Inter-Domain Routing

Example:

192.168.1.0/24

24 bits = Network

8 bits = Host

---

## Quick Memory Trick

255 = 8 bits

11111111

Every 1 represents network portion.

## Interview Question

Q: What is the purpose of a subnet mask?

Answer:
A subnet mask separates network bits and host bits within an IP address.
