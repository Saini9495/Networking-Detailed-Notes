# Network Devices

## Introduction

Network devices are hardware components that help establish communication between computers and other devices in a network.

They are responsible for:

- Forwarding data
- Connecting devices
- Managing traffic
- Providing internet access
- Securing communication

---

# 1. Hub

A Hub is a basic networking device that connects multiple devices in a network.

It works at:

OSI Layer 1 (Physical Layer)

---

## How Hub Works?

When a device sends data to a Hub:

- Hub receives the data
- Broadcasts it to every connected device
- Every device receives the frame
- Only the intended receiver processes it

---

## Advantages

- Cheap
- Easy to install

## Disadvantages

- Creates network congestion
- Low security
- Inefficient communication

---

# 2. Switch

A Switch is an intelligent networking device that connects devices within a LAN.

It works at:

OSI Layer 2 (Data Link Layer)

Some advanced switches work at Layer 3.

---

## How Switch Works?

Switch uses MAC Addresses.

When data arrives:

1. Switch learns MAC addresses
2. Stores them in MAC Address Table
3. Sends data only to intended device

---

## Advantages

- Faster than Hub
- Reduced traffic
- Better security
- Efficient communication

## Disadvantages

- More expensive than Hub

---

# 3. Router

A Router connects different networks together.

It works at:

OSI Layer 3 (Network Layer)

---

## Main Function

Router forwards packets using IP Addresses.

Example:

Home Router connects:

- Home Network
- Internet

---

## How Router Works?

1. Receives packet
2. Checks destination IP
3. Uses Routing Table
4. Sends packet to next network

---

## Advantages

- Connects multiple networks
- Provides internet access
- Supports security features

---

# 4. Modem

Modem stands for:

Modulator + Demodulator

---

## Function

Converts:

Digital Signals ↔ Analog Signals

Used by Internet Service Providers (ISPs).

---

## Example

Your ISP provides internet through:

- Fiber
- DSL
- Cable

The modem converts incoming signals into data your computer can understand.

---

# 5. Access Point (AP)

An Access Point provides wireless connectivity.

It allows WiFi devices to connect to a wired network.

---

## Example

Office WiFi

Devices connect wirelessly through an Access Point.

---

# 6. Firewall

A Firewall is a security device.

It monitors and filters incoming and outgoing network traffic.

---

## Purpose

- Block unauthorized access
- Allow trusted traffic
- Protect internal network

---

# 7. Gateway

A Gateway connects two different networks using different protocols.

---

## Example

A company network communicating with a cloud network.

Gateway acts as a translator.

---

# 8. Repeater

A Repeater regenerates weak signals.

Used when signal strength decreases over long distances.

---

## Benefits

- Extends network range
- Improves signal quality

---

# 9. Bridge

A Bridge connects multiple LAN segments.

It filters traffic based on MAC addresses.

---

# Comparison Table

| Device | OSI Layer | Uses |
|----------|-----------|----------|
| Hub | Layer 1 | Broadcast Data |
| Switch | Layer 2 | Forward Frames |
| Router | Layer 3 | Route Packets |
| Firewall | Layer 3-7 | Security |
| Access Point | Layer 2 | Wireless Access |
| Modem | Layer 1 | Signal Conversion |
| Repeater | Layer 1 | Signal Boosting |
| Bridge | Layer 2 | Connect LAN Segments |

---

# Summary

Network devices form the backbone of communication systems. Understanding their functions is essential for networking, cloud computing, cybersecurity, and DevOps.
