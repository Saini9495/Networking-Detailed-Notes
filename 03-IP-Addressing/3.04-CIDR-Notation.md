# CIDR Notation

CIDR stands for:

Classless Inter-Domain Routing

CIDR is used to define subnet masks efficiently.

Example:

192.168.1.0/24

The number after "/" represents Network Bits.

## CIDR Formula

Hosts = 2^(32 - Prefix Length) - 2

Example:

/24

Hosts:

2^(32-24)-2

2^8-2

256-2

254 Hosts

## Why CIDR?

Benefits:

- Better IP Utilization
- Efficient Routing
- Reduced Routing Tables
- Supports Subnetting
