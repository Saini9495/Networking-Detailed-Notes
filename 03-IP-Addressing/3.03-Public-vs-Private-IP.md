# Public IP vs Private IP

Every device on a network uses an IP address.

IP addresses are categorized into:

1. Public IP
2. Private IP

---

# Public IP Address

A Public IP is globally unique and accessible over the Internet.

Assigned by:

Internet Service Provider (ISP)

Examples:

8.8.8.8

1.1.1.1

## Characteristics

- Globally Unique
- Internet Accessible
- Assigned by ISP

## Uses

- Hosting Websites
- Cloud Servers
- Remote Access

---

# Private IP Address

Private IP addresses are used inside internal networks.

They are not directly accessible from the Internet.

## Private IP Ranges

### Class A

10.0.0.0 - 10.255.255.255

### Class B

172.16.0.0 - 172.31.255.255

### Class C

192.168.0.0 - 192.168.255.255

## Characteristics

- Internal Use
- Not Routable on Internet
- Reusable Across Networks

## Example

Home WiFi:

Laptop:

192.168.1.10

Phone:

192.168.1.20

Router:

192.168.1.1

---

# Public vs Private IP

| Feature | Public IP | Private IP |
|-----------|-----------|-----------|
| Internet Access | Yes | No |
| Unique Globally | Yes | No |
| Assigned By | ISP | Router |
| Cost | Usually Paid | Free |
