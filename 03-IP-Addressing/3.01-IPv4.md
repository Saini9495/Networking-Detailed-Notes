# IPv4 (Internet Protocol Version 4)

IPv4 is the fourth version of the Internet Protocol used to identify devices on a network.

Every device connected to a network is assigned a unique IP Address.

Example:

192.168.1.10

## IPv4 Structure

IPv4 consists of 32 bits divided into 4 octets.

Example:

192.168.1.10

Binary Representation:

11000000.10101000.00000001.00001010

Each octet contains 8 bits.

Total:

8 × 4 = 32 Bits

## IPv4 Format

xxx.xxx.xxx.xxx

Range:

0.0.0.0 to 255.255.255.255

## Components of IPv4

An IPv4 address contains:

1. Network Portion
2. Host Portion

Example:

192.168.1.10/24

Network ID:

192.168.1.0

Host ID:

10

## Classes of IPv4

### Class A

Range:

1.0.0.0 - 126.255.255.255

Default Mask:

255.0.0.0

Supports:

16 Million Hosts

Example:

10.0.0.1

---

### Class B

Range:

128.0.0.0 - 191.255.255.255

Default Mask:

255.255.0.0

Supports:

65,534 Hosts

Example:

172.16.0.1

---

### Class C

Range:

192.0.0.0 - 223.255.255.255

Default Mask:

255.255.255.0

Supports:

254 Hosts

Example:

192.168.1.1

---

### Class D

Used for Multicasting

Range:

224.0.0.0 - 239.255.255.255

---

### Class E

Reserved for Research

Range:

240.0.0.0 - 255.255.255.255

## Advantages

- Simple
- Widely Used
- Easy Configuration

## Limitations

- Limited Addresses
- Address Exhaustion
- Less Secure

## Real Life Example

Think of an IP address like a house address.

If someone wants to deliver a package, they need your house address.

Similarly, data packets need an IP address to reach your device.
