# DNS (Domain Name System)

## What is DNS?

DNS (Domain Name System) is the phonebook of the Internet.

Hum humans website names yaad rakh sakte hain:

google.com
github.com
amazon.com

Lekin computers IP addresses samajhte hain:

142.250.193.78

DNS domain name ko IP address me convert karta hai.

---

## Real-Life Example

Suppose tumhare phone me:

Rohit = 9876543210

Tum Rohit search karte ho, phone automatically number nikal deta hai.

Similarly:

google.com → DNS → 142.250.193.78

---

## Why DNS is Needed?

Without DNS:

https://google.com

ki jagah

http://142.250.193.78

type karna padta.

DNS internet ko user-friendly banata hai.

---

## DNS Hierarchy

DNS follows a hierarchical structure.

Root Domain (.)
    |
    ├── .com
    ├── .org
    ├── .net
    |
    └── google.com

---

## Components of DNS

### 1. DNS Resolver

User ki request receive karta hai.

Example:

ISP DNS Server
Google DNS
Cloudflare DNS

---

### 2. Root Name Server

Top-level DNS server.

Ye batata hai:

".com" ke server kahan hain.

---

### 3. TLD Server

TLD = Top Level Domain

Examples:

.com
.org
.net
.io

---

### 4. Authoritative DNS Server

Actual domain records store karta hai.

Example:

google.com ka actual IP.

---

## DNS Resolution Process

Step 1:
Browser → google.com

Step 2:
DNS Resolver query karta hai.

Step 3:
Root Server se poochta hai.

Step 4:
Root → .com server batata hai.

Step 5:
TLD → Authoritative Server batata hai.

Step 6:
Authoritative Server IP return karta hai.

Step 7:
Browser website open karta hai.
