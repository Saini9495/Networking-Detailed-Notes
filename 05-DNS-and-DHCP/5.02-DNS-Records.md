# DNS Records

DNS records are instructions stored in DNS servers.

They define how a domain behaves.

---

## A Record

Maps Domain → IPv4 Address

Example:

google.com → 142.250.193.78

Record:

A
google.com
142.250.193.78

---

## AAAA Record

Maps Domain → IPv6 Address

Example:

AAAA
google.com
2404:6800:4009:80b::200e

---

## CNAME Record

Alias record.

Maps one domain to another domain.

Example:

blog.example.com

→ example.com

Record:

CNAME
blog
example.com

---

## MX Record

Mail Exchange Record.

Used for Email Servers.

Example:

gmail.com

MX
mail.google.com

---

## TXT Record

Stores text information.

Used for:

- Domain Verification
- SPF
- DKIM
- Security

Example:

TXT
v=spf1 include:_spf.google.com ~all

---

## NS Record

Name Server Record.

Defines authoritative DNS server.

Example:

ns1.example.com
ns2.example.com

---

## PTR Record

Reverse DNS Lookup.

IP → Domain Name

Example:

142.250.193.78

→ google.com

---

## SRV Record

Service Location Record.

Used for:

- SIP
- VoIP
- Microsoft Services

---

## SOA Record

Start Of Authority Record.

Contains:

- Domain owner
- Serial number
- Refresh interval

---

## Most Important Records

1. A
2. AAAA
3. CNAME
4. MX
5. TXT
6. NS

These are heavily used in Cloud and DevOps.
