# DNS Lookup

DNS Lookup is the process of finding the IP address associated with a domain.

Example:

google.com

↓

142.250.193.78

---

## Forward Lookup

Domain → IP

Example:

google.com

↓

142.250.193.78

---

## Reverse Lookup

IP → Domain

Example:

142.250.193.78

↓

google.com

Uses PTR Record.

---

## DNS Lookup Tools

### nslookup

Windows/Linux

Command:

nslookup google.com

Output:

Name: google.com
Address: 142.250.193.78

---

### dig

Linux

Command:

dig google.com

Shows detailed DNS information.

---

### host

Linux

Command:

host google.com

Output:

google.com has address 142.250.193.78

---

## Check MX Records

dig gmail.com MX

or

nslookup -type=MX gmail.com

---

## Check TXT Records

dig google.com TXT

---

## Check Name Servers

dig google.com NS

---

## Common Errors

### NXDOMAIN

Domain not found.

### SERVFAIL

DNS server failure.

### Timeout

DNS response not received.

## Interview Questions

Q1. What is DNS Lookup?

Process of converting domain name into IP address.

Q2. Difference between Forward and Reverse Lookup?

Forward:
Domain → IP

Reverse:
IP → Domain
