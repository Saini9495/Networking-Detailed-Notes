# Switch

## What is a Switch?

A Switch is a Layer 2 networking device that connects multiple devices within a network.

It uses MAC Addresses to send data to the correct destination.

---

## Why Switch is Used?

Without Switch:

Data would be sent to every device.

With Switch:

Data reaches only intended device.

---

## How Switch Works?

1. Device sends frame
2. Switch learns MAC Address
3. Stores MAC in MAC Table
4. Sends frame to correct port

---

## MAC Address Table

Example:

| MAC Address | Port |
|------------|------|
| AA:11 | 1 |
| BB:22 | 2 |
| CC:33 | 3 |

Switch uses this table to forward frames.

---

## Types of Switches

### Unmanaged Switch

- Plug and Play
- No Configuration

### Managed Switch

- Supports VLAN
- Security Features
- Monitoring

---

## Layer 2 Switch

Uses MAC Addresses.

---

## Layer 3 Switch

Can perform routing.

Uses:

- IP Addresses
- Routing Functions

---

## Advantages of Switch

- Fast Communication
- Reduced Collisions
- Better Performance
- Improved Security
