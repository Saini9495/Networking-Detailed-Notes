# MAC Address

## What is MAC Address?

MAC stands for:

Media Access Control

It is a unique physical address assigned to a network interface card (NIC).

---

## MAC Address Example

00:1A:2B:3C:4D:5E

Length:

48 Bits

or

6 Bytes

---

## MAC Address Structure

Example:

00:1A:2B : 3C:4D:5E

First Half:
Manufacturer ID (OUI)

Second Half:
Device Identifier

---

## Why MAC Address is Important?

Switches use MAC addresses to:

- Identify devices
- Forward frames
- Build MAC Table

---

## MAC vs IP Address

| MAC Address | IP Address |
|------------|------------|
| Physical Address | Logical Address |
| Permanent | Can Change |
| Layer 2 | Layer 3 |
| Local Communication | Network Communication |

---

## Find MAC Address in Linux

```bash
ip link
```

or

```bash
ifconfig
```

Output:

08:00:27:12:34:56

---

## Find MAC Address in Windows

```cmd
ipconfig /all
```

---

## Important Facts

- Every NIC has a MAC Address
- Used in LAN Communication
- Works at Data Link Layer
- Helps switches identify devices
