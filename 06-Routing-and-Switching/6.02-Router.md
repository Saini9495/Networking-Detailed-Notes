# Router

## What is a Router?

A Router is a Layer 3 networking device that connects different networks and forwards packets based on IP addresses.

---

## Main Functions of Router

### 1. Routing

Selects the best path for data.

### 2. Packet Forwarding

Forwards packets between networks.

### 3. Network Segmentation

Separates different networks.

### 4. Internet Connectivity

Connects local network to Internet.

---

## How Router Works?

Example:

Laptop IP:
192.168.1.10

Website IP:
142.250.183.78

Steps:

1. Laptop sends request
2. Router receives packet
3. Router checks routing table
4. Router forwards packet to ISP
5. Packet reaches destination

---

## Components of Router

### CPU

Processes routing decisions.

### RAM

Stores routing tables.

### Interfaces

Connects networks.

Examples:

- Ethernet
- Fiber
- Wireless

---

## Types of Routers

### Home Router

Used in homes.

### Enterprise Router

Used in companies.

### Core Router

Used by ISPs.

### Edge Router

Connects organization to Internet.

---

## Router vs Switch

| Feature | Router | Switch |
|----------|----------|----------|
| Layer | Layer 3 | Layer 2 |
| Uses | IP Address | MAC Address |
| Connects | Networks | Devices |
| Broadcast Domain | Separate | Same |

---

## Common Router Commands

Linux:

View Routing Table:

```bash
ip route
```

View Interfaces:

```bash
ip addr
```

Check Connectivity:

```bash
ping google.com
```
