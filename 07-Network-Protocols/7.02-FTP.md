# FTP (File Transfer Protocol)

## What is FTP?

FTP stands for File Transfer Protocol.

It is used to transfer files between client and server over a network.

---

## Uses

- Upload Website Files
- Download Files
- Backup Data
- Server Management

---

## Default Ports

| Mode | Port |
|--------|--------|
| Control Connection | 21 |
| Data Connection | 20 |

---

## FTP Architecture

Client <-----> FTP Server

---

## FTP Working

1. Client connects to FTP server.
2. Authentication happens.
3. User uploads/downloads files.
4. Session ends.

---

## FTP Commands

### LIST

View files.

### GET

Download file.

### PUT

Upload file.

### DELETE

Delete file.

---

## Types of FTP

### Active FTP

Server initiates data connection.

### Passive FTP

Client initiates data connection.

Most commonly used today.

---

## Problems with FTP

- No Encryption
- Credentials sent in plain text
- Vulnerable to attacks

---

## Alternatives

### SFTP

FTP over SSH

Port: 22

### FTPS

FTP over SSL/TLS

---

## DevOps Usage

Used for:

- Legacy Applications
- File Transfers
- Backup Servers

Modern systems prefer SFTP.
