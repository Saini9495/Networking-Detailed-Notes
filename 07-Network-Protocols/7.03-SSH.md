# SSH (Secure Shell)

## What is SSH?

SSH stands for Secure Shell.

SSH is a secure protocol used to remotely access servers.

---

## Why SSH?

Before SSH, Telnet was used.

Problem:

- No Encryption
- Passwords visible in plain text

SSH solved this problem.

---

## Default Port

| Protocol | Port |
|----------|------|
| SSH | 22 |

---

## SSH Features

- Encrypted Communication
- Secure Login
- File Transfer
- Remote Command Execution

---

## SSH Architecture

Client ---- SSH ---- Server

---

## SSH Authentication

### Password Authentication

Username + Password

### Key-Based Authentication

More secure.

Uses:

- Public Key
- Private Key

---

## Generate SSH Key

```bash
ssh-keygen -t rsa -b 4096
```

---

## Connect to Server

```bash
ssh ubuntu@192.168.1.10
```

---

## Copy Files

### SCP

```bash
scp file.txt ubuntu@server:/home/ubuntu
```

---

## SSH Services

```bash
systemctl status ssh
```

---

## DevOps Usage

Used in:

- Linux Servers
- AWS EC2
