# SNMP (Simple Network Management Protocol)

## What is SNMP?

SNMP is used for monitoring and managing network devices.

---

## Devices Managed by SNMP

- Routers
- Switches
- Servers
- Firewalls
- Printers

---

## SNMP Components

### SNMP Manager

Monitoring System

Examples:

- Nagios
- Zabbix
- PRTG

---

### SNMP Agent

Software running on device.

Collects information.

---

### MIB

Management Information Base

Database containing device information.

---

## Default Ports

| Protocol | Port |
|----------|------|
| SNMP | 161 |
| SNMP Trap | 162 |

---

## SNMP Versions

### SNMPv1

Basic version.

### SNMPv2c

Improved performance.

### SNMPv3

Most secure.

Supports:

- Authentication
- Encryption

---

## SNMP Operations

### GET

Retrieve information.

### SET

Modify information.

### TRAP

Send alert automatically.

---

## Example Metrics

- CPU Usage
- Memory Usage
- Interface Traffic
- Disk Usage
- Network Errors
