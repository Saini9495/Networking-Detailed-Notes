# ICMP (Internet Control Message Protocol)

## What is ICMP?

ICMP is used for error reporting and network diagnostics.

It works at the Network Layer.

---

## Uses

- Connectivity Testing
- Error Reporting
- Network Troubleshooting

---

## Common ICMP Messages

### Echo Request

Ping Request

### Echo Reply

Ping Response

### Destination Unreachable

Destination not reachable.

### Time Exceeded

TTL expired.

---

## Ping Command

```bash
ping google.com
```

---

## Traceroute

```bash
traceroute google.com
```

---

## ICMP Features

- No Data Transfer
- Diagnostic Protocol
- Lightweight
