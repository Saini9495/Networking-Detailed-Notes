# SMTP, POP3 and IMAP

These protocols are used in Email Communication.

---

# SMTP

## Simple Mail Transfer Protocol

Used for sending emails.

Example:

User → Gmail Server → Receiver Mail Server

---

## SMTP Ports

| Port | Usage |
|--------|--------|
| 25 | Default |
| 587 | Secure Submission |
| 465 | SSL/TLS |

---

# POP3

## Post Office Protocol Version 3

Used to receive emails.

Downloads emails from server to local device.

---

## POP3 Port

| Port | Usage |
|--------|--------|
| 110 | Default |
| 995 | Secure |

---

## POP3 Characteristics

- Downloads emails
- Removes emails from server
- Offline access

---

# IMAP

## Internet Message Access Protocol

Used for receiving emails.

Keeps emails on server.

---

## IMAP Port

| Port | Usage |
|--------|--------|
| 143 | Default |
| 993 | Secure |

---

## IMAP Characteristics

- Synchronization across devices
- Emails remain on server
- Better for modern systems

---

## POP3 vs IMAP

| Feature | POP3 | IMAP |
|-----------|---------|---------|
| Multi Device Sync | No | Yes |
| Emails on Server | No | Yes |
| Offline Access | Yes | Limited |
| Modern Usage | Less | More |

---
